I was shown an interesting way to get data into the database from the controller.

In your Angular controller, append the property (in the schema name) to the directive that appears in your view. The proter that I am appending here is '.creator.'

$scope.currentUser = userID;
$scope.newRap.creator = $scope.currentUser;

Next, add the object that goes into the directive into your new factory model('newRap'), like you would otherwise do.

    var userRap = new rapFactory.model(this.newRap)
        userRap.$save(function(returnData) {
        ..
        }

This is passed into your backend controller as normal:

    var apiController = {
        createRap: function(req, res) {
            var newDbRap = new Models.Rap(req.body)
            newDbRap.save(function(err, doc) {
    })
}

In my schema, 'creator' is ready to receive the data.

    var rapSchema = mongoose.Schema({
    rap     : String,
    date    : String,
    creator : String
    })

Check your database, it's there! This option seems to be most appropriate when only dealing with a couple of pieces of data.
